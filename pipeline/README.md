# Pipeline

This folder holds the work in progress.

- `ideas/`: weekly source reports and topic ideas.
- `projects/`: one folder per video package. This is the preferred workflow.
- `scripted/`: approved scripts; scripts include clip/interview/graphic cues.
- `recorded/`: Dean's voiceover files.
- `editing/`: temporary working files.
- `published/`: finished projects after upload.

Normal order: ideas first, project package second, script and metadata third,
script-based clips fourth, recording fifth, assembly last.
