# Voice

This folder stores Dean's voice examples.

- `scripts/`: reusable scripts and writing references.
- `transcripts/`: transcripts of prior Dean videos, organized by year.
- `_scratch/`: temporary transcript or analysis work.

The scripts read transcripts recursively, so year folders are safe.

New voiceovers for client videos should go inside the matching
`pipeline/projects/<video>/voiceover/` folder. `pipeline/recorded/` remains a
legacy fallback.
